# Task 6 — House Price Prediction

## Objective
To build a machine learning model that predicts house prices using important property features.

## Dataset Used
House Price Prediction Dataset

## Model Used
- Linear Regression

## Evaluation Metrics
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)

## Key Findings
- Living area, bathrooms, and bedrooms strongly affected house prices
- Linear Regression performed well for prediction
- Actual vs Predicted plots showed reliable performance
