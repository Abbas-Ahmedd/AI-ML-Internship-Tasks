# AI/ML Engineering Internship Tasks

This repository contains my completed AI/ML Engineering Internship Tasks for DevelopersHub Corporation.

As part of the internship requirements, at least 3 out of 6 tasks were required to be completed. These projects focus on data analysis, machine learning, model evaluation, and real-world problem solving using Python and Jupyter Notebook.

---

# Completed Tasks

## Task 1 — Exploring and Visualizing the Iris Dataset

### Objective
To load, inspect, and visualize the Iris dataset to understand feature distributions, relationships, and outliers.

### Skills Used
- Pandas
- Seaborn
- Matplotlib
- Data Visualization
- Exploratory Data Analysis (EDA)

### Key Work
- Dataset loading and inspection
- Scatter Plot
- Histograms
- Box Plot
- Final insights and conclusion

---

## Task 3 — Heart Disease Prediction

### Objective
To build a machine learning model that predicts whether a person is at risk of heart disease using medical data.

### Skills Used
- Logistic Regression
- Classification
- Confusion Matrix
- ROC Curve
- Feature Importance Analysis

### Key Work
- Data cleaning
- EDA
- Model training
- Accuracy evaluation
- ROC-AUC analysis
- Final insights and conclusion

---

## Task 6 — House Price Prediction

### Objective
To predict house prices using important property features such as bedrooms, bathrooms, area, and floors.

### Skills Used
- Linear Regression
- Regression Modeling
- MAE
- RMSE
- Data Preprocessing

### Key Work
- Feature selection
- EDA
- Model training
- Price prediction
- Actual vs Predicted visualization
- Final insights and conclusion
