# Task 3 — Heart Disease Prediction

## Objective
To build a machine learning model that predicts whether a person is at risk of heart disease based on medical data.

## Dataset Used
Heart Disease UCI Dataset

## Models Used
- Logistic Regression
- Decision Tree (optional if tested)

## Evaluation Metrics
- Accuracy Score
- Confusion Matrix
- Classification Report
- ROC Curve
- AUC Score

## Key Findings
- Logistic Regression performed well in classification
- Chest pain type, cholesterol, oldpeak, and max heart rate were strong predictors
- The model showed reliable prediction performance
